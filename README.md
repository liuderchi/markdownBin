# markdownBin

  - simple markdown collaborative editor powered by meteor and react

![screenshot][screenshot]


## Env

  - install meteor
      - `curl https://install.meteor.com/ | sh`
  - install npm dependencies
      - `npm install`

## Run

  - `$ meteor run`
      - server will listen at port 3000 by default
